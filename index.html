Esto es una página web muy básica. En ella vamos a practicar las etiquetas más importantes y cómo cambiar los estilos con hojas css.

Etiquetas y divisiones

Todas las páginas tienen que estar estructuradas en partes. Vamos a usar párrafos (etiqueta <p>) y divisiones (etiqueta <div>). Ambas tienen una etiqueta para abrir y otra para cerrar (con barra diagonal).

El párrafo es la unidad básica. Todo párrafo debería usar estas etiquetas. La división es algo paralelo, para crear "bloques", y puede tener dentro párrafos, pero no al revés. La división puede anidarse.

Ejemplos:

  <div>hola</div>
</div>

Cómo funcionan los estilos

Cuando se carga la página web, la cabeza ("head") carga el archivo css/estilos.css. Cada etiqueta puede tener en este archivo una definición de estilo. Cada etiqueta también puede tener un parámetro class, cuya definición también (debe) estar dentro del archivo de estilos (con el nombre precedido por un punto).

Veamos un ejemplo:

.cita {
  font-style:italic;
  color:azure;
}

Se vería:

Este párrafo tiene el estilo "cita" aplicado. Le he indicado eso escribiendo <p class = "cita"> en la etiqueta en la que comienza el párrafo.
